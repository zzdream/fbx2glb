Linux x86_64：请在此目录放入以下可执行文件后再执行 pnpm build:linux

1. fbx2gltf
   - 下载：https://github.com/facebookincubator/FBX2glTF/releases 中的 FBX2glTF-linux-x64
   - 重命名为 fbx2gltf 放入此目录

2. gltfpack
   - 下载：https://github.com/zeux/meshoptimizer/releases
   - 选择 Linux x64 压缩包，解压后将 gltfpack 放入此目录

放入后执行：chmod +x fbx2gltf gltfpack
